# 更新日志

## [3.1.1] - 2026-04-19

### 打包优化
- 调整 `pgt.spec` 为更偏启动速度和稳定性的配置
- 保持目录分发模式，避免 `one-file` 带来的冷启动解包开销
- 关闭 `UPX` 压缩，并启用 `noarchive=True`，改善 Windows 打包后的启动体验

### 项目清理
- 删除未再使用的 `PGT-Desktop.ui`
- 删除未参与主程序运行的 `wordmargin.py`
- 继续清理历史遗留文件，降低项目维护成本

## [3.1.0] - 2026-04-19

### 架构重构
- 将原本集中在 `app.py` 中的逻辑拆分为 `ui`、`services`、`models`、`utils` 等模块
- 新增 `main.py` 作为主入口，保留 `app.py` 作为兼容入口
- 引入 `GenerationTask` 数据模型，统一管理文档生成参数
- 将模板选择、图片处理、布局整理、Word 导出分别封装到独立服务中
- 新增 `LayoutService` 与 `WordExporter`，将排版准备与 Word 输出解耦

### 性能优化
- 将文档生成与图片处理迁移到后台线程，避免阻塞主界面
- 去除常驻日期日历控件，改用更轻量的日期输入方式，显著提升启动速度
- 优化图片处理流程，减少不必要的磁盘中转
- 调整打包配置为目录分发模式，改善应用启动体验

### 界面优化
- 重构主界面布局，整理输入区、状态区和按钮区的层级
- 新增生成状态提示和进度条反馈
- 优化按钮状态切换逻辑，生成期间禁用关键操作按钮
- 调整样式细节，统一输入框、按钮、状态区的视觉表现
- 移除 PDF 导出选项，简化主界面交互

### 开发体验
- 补充 `README.md`，新增开发运行说明与打包说明
- 清理构建产物、缓存文件和运行日志
- 更新 `.gitignore`，忽略虚拟环境、缓存、构建目录和日志文件
- 移除 `docx2pdf` 依赖及其打包配置

## [3.0.1] - 2025-04-06

### 新增功能
- 增加可选生成PDF文档的选项
- 增加生成后自动打开文档的功能

### 界面优化
- 调整了按钮和输入框的布局，调整组件间距和尺寸

### 性能改进
- 统一裁剪图片尺寸为 6.4cm * 4.8cm
- 提高了图片清晰度，将DPI从96调整为240
- 优化了图片压缩和处理逻辑

### 其他改进
- 优化了错误处理机制，提供更详细的错误提示
- 改进了代码结构和注释，提高可维护性
